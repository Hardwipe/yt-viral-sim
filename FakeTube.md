# 📢 Release v2 – FakeTube Viral Simulator  
**Website:** https://www.fake-tube.com

FakeTube returns with **Version 2**, bringing a more immersive and realistic viral simulation experience for creators, developers, and anyone curious about how online virality unfolds.

Originally launched as a playful tool to simulate YouTube growth, FakeTube has evolved into a **fully interactive viral dashboard** with expanded analytics, simulation controls, and a more dynamic user experience.

---

# 🚀 Features

## 🎬 Video Virality Simulator

Upload any video and simulate it going viral with a fully interactive growth dashboard.

FakeTube dynamically generates:

- Views  
- Likes  
- Subscribers  
- Comments  

to replicate the experience of watching a video gain traction in real time.

---

# 📊 Live Analytics Dashboard

FakeTube includes a **real-time analytics graph** tracking:

- Views  
- Likes  
- Subscribers  
- Comments  

The chart updates continuously as the simulation runs, showing how engagement evolves over time.

Analytics continue running **in the background**, even when the chart is locked.

---

# 🔒 Unlockable Analytics System

A new **analytics unlock system** simulates premium creator dashboards.

When locked:

- Analytics are blurred  
- Simulation continues running  
- A live **Algorithm Confidence Meter** predicts performance  

Unlocking analytics reveals the full real-time graph.

⚠️ Disclaimer displayed:

> Analytics shown are simulated and may not reflect actual YouTube performance.

---

# 💬 Advanced Comment System

The comment feed simulates realistic engagement using multiple comment styles:

- Viral hype comments  
- Early viewer comments  
- Meme-style comments  
- Timestamp reactions  
- Bot-style promotional comments  
- Viewer reaction comments  
- Spam-style comments  

Each comment includes:

- Generated usernames  
- Handles  
- Avatars  
- Like counts  
- Relative timestamps  

---

# 🤖 Bot Raid System

Trigger a **Bot Raid** event to simulate a sudden spike in comments.

Bot raids inject **50–100 comments instantly**, creating a realistic viral engagement burst while updating analytics simultaneously.

---

# 🚀 Viral Boost Events

Manually trigger **viral growth spikes** that increase:

- Views  
- Likes  
- Subscribers  

This simulates algorithmic boosts and trending discovery moments.

---

# 🔔 Algorithm Event Notifications

FakeTube now generates simulated algorithm alerts including:

- Viral spike detection  
- Engagement velocity increases  
- Recommendation system signals  
- Audience growth spikes  
- Trending detection  

These appear as **live notification toasts** during simulation.

---

# 🎛️ Expanded Simulation Controls

Creators now have fine-grained control over simulation behavior.

You can:

- Adjust speed of views, likes, subscribers, and comments  
- Pause or resume specific engagement metrics  
- Pause or resume the entire simulation  
- Reset the simulation instantly  
- Trigger bot raids or viral boosts  

---

# 🌙 Dark Mode / Light Mode

A new UI theme toggle allows switching between:

- Dark Mode  
- Light Mode  

for improved dashboard readability and creator-style aesthetics.

---

# 📊 Algorithm Confidence Meter

FakeTube estimates how strongly the algorithm is favoring the video.

The **Algorithm Confidence Score** dynamically updates based on:

- Views  
- Likes  
- Subscribers  
- Comments  

This provides a playful simulation of how recommendation systems might evaluate performance.

---

# 🧠 Internal Event Logging System

FakeTube now logs important simulation events including:

- Simulation start / pause / stop  
- Viral boosts  
- Bot raids  
- Analytics unlock events  
- Comment feed resets  
- Mode changes  

Logs are stored locally in the browser for debugging and development.
